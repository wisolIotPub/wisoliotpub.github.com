        SPB002E

        SPB002E is a dual band 2x2 WLAN and Bluetooth module for mobile devices application. Broadcom’s BCM4354 is integrated.

        Features

          - Dimension : 9.2x5.7x1.2mm
          - Package : BGA 72pins 
          - Temperature Range : -30℃ ~ +85℃
          - Supply Voltage : 3.2V to 4.8V
          - Interface : SDIO3.0 for WLAN, UART for Bluetooth
          - WLAN Spec : 802.11a/b/g/n/ac, dual band 2x2 MIMO
          - BT Spec : BT3.0 and 4.0BLE, Class1.5
           

            Applications

              - Mobile handsets
              - Personal computers
              - IoT and Multimedia
